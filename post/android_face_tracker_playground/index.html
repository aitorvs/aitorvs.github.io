Android Face-Tracking Playground

Sep 2, 2016
android mobile java' programming development technology
1 min read

In late 2015, Google published the Mobile Vision API as part of the Google Play Services. That API comprises several packages for face detection, barcode scan and text recognition. Out of them, the most compelling to me is the face detection API, maybe because is the most user-interactive one.

I’ve recently published an article on Medium that discusses the basics to get started on face detection. You’ll be pleasantly surprise about how simple that is, so just go! read the article, and create awesome things yourself!

If you want to skip the article and get craking, get the full source code of the sample app from my face-tracking-sample GitHub project or; download it from Google Play beta channel.
